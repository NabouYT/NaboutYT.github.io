Aly - Assistente Virtual Inteligente

Desenvolvimento de um assistente virtual em Python com capacidades de conversação inteligente e navegação web em tempo real. A Aly utiliza um modelo de linguagem avançado (Llama3 via Ollama) para interpretar comandos e, quando necessário, executa web scraping (com Selenium) para buscar informações dinâmicas como hora atual e notícias. Este projeto demonstra minhas habilidades em **integração de IA, web scraping, automação, lógica de programação e tratamento de dados**.
